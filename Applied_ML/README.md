# Medical Question Answering System

## Project Overview
This project implements a semantic search-based Medical Question Answering System using the MedQuAD dataset.

## Dataset
Dataset used: MedQuAD (Medical Question Answering Dataset)

The dataset contains medical questions and corresponding answers.

## Approach

1. Load the MedQuAD dataset.
2. Convert question-answer pairs into vector embeddings using Sentence Transformer.
3. Build a FAISS similarity search index.
4. Retrieve relevant answers based on user queries.

## Technologies Used

- Python
- Pandas
- Sentence Transformers
- FAISS
- NumPy
